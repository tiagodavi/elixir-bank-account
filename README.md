### Starting API

* open a terminal
* $ `cd api`
* $ `mix deps.get`
* $ `mix test`
* $ `mix phx.server`

### Starting Site

* open a terminal
* $ `cd site`
* $ `npm install`
* $ `npm start`

Make sure both services are running in different terminals.

Now you can visit [`localhost:3000`](http://localhost:3000) from your browser.
